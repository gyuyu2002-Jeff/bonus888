📖 互盛營業考核與獎金計算平台 使用說明書

🎯 平台設計宗旨

本計算平台專為互盛（HUXEN）營業同仁與主管設計，旨在協助營業考核試算以及日常成交獎金的估算。透過整合級距評分表、提供晉升.合格門檻條件，提供一鍵式的判定與回饋，協助同仁提早規劃業績並掌握薪酬權益。

🚀 系統使用五步驟

1. 身分與考評設定 選擇您的「事業部類別」（OA / CA）、「職務身分」（營業同仁 / 主任）與「職級」，並根據考評目的設定評核類型。
2. 確認保級門檻 設定身分後，左方「保級免遭調降合格門檻」卡片會即時呈現出該職級在此考核期內對應的三大保級方案的個人與團隊業績要求。
3. 逐月輸入實際實績 在表格中填寫您每個月的「實際」值。上方的「目標」值會自動帶出。輸入時系統會於背景**自動存檔**，且畫面不會跳動。
4. 一鍵計算判定與預估 輸入完畢後點擊「開始計算考核」。系統將呈現總分與合格檢核，若填寫未滿期，還會顯示**「未來月份業績台數預估」**。

💾 資料備份、還原與隱私安全

為了最大化便利性與資料安全，平台採用以下備份機制：

  • 瀏覽器自動存檔：只要您輸入任何資料，系統便會自動將其保存在您的本機瀏覽器（LocalStorage）中。即使您關閉瀏覽器或重新整理，資料也不會遺失。
  • JSON 檔案備份與還原：點擊「匯出備份」即可將目前輸入的所有考評與獎金數據下載為一個備份檔案（.json）。下次使用時點選「匯入備份」即可完美復原。
  • 完美支援檔案改名：為了方便檔案管理，您可以自由更改匯出的備份檔案名稱（例如改為 `2026-07_Jeff特考.json`），系統在匯入時依然能夠完美解析資料。
  • 100% 隱私保護：本系統為純前端應用程式，所有計算與存檔皆在您自己的電腦瀏覽器中完成，**絕不會上傳任何個資或業績數據至任何外部伺服器**，請放心使用。
